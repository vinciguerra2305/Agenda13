Meu Portal de Entretenimento Digital

Projeto simples em HTML criado para exibir filmes, séries e conteúdos de entretenimento com imagens, títulos e descrições.

Sobre o Projeto

Este projeto foi desenvolvido utilizando apenas HTML e CSS básico.

O objetivo é praticar:

Estruturação de páginas HTML
Inserção de imagens
Organização visual com CSS
Criação de páginas de entretenimento

Tecnologias Utilizadas
HTML5
CSS3
